# Lazy To Cook - Android Application

Android mobile app for the Lazy To Cook food delivery platform.

## Tech Stack

- **Language:** Kotlin
- **UI:** Jetpack Compose
- **Min SDK:** 24 (Android 7.0)
- **Target SDK:** 36

## Prerequisites

- Android Studio
- JDK 11 or higher

## Setup

1. Open the project in Android Studio
2. Sync Gradle dependencies
3. Run the app on an emulator or physical device

## Build

```bash
./gradlew build
```

## Run Tests

```bash
./gradlew test
```
